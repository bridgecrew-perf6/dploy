Ansible deployment toolbox

### Intallation

```bash
curl -ssf -L https://raw.githubusercontent.com/ca-gip/dploy/master/install.sh | bash
```

Don't forget to add completion

```
echo "source <(dploy completion bash)" >> ~/.bashrc
```
